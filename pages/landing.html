Optional Tor Routing

The backend and routing docs still support the PHANTOM path through Tor and hardened egress controls when anonymity layering is required.

DNS Hardening

Resolver-level filtering, leak prevention, and ad/tracker blocking remain intact while the landing experience now matches the Phantom Vault visual system.

Threat Monitoring

Real-time monitoring, attack detection, and live dashboard metrics remain available from the existing backend and dashboard routes.

Defense in Depth

WireGuard, DNS filtering, fail2ban, admin controls, and browser-side protections work as layered defenses rather than isolated marketing claims.

Zero-Log Architecture

No traffic logging, no DNS query history, and no hidden session resets. Trial metering remains explicit and anchored to actual usage state.
