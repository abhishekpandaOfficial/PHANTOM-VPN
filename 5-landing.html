First-time onboarding

This is the real user flow in PHANTOM today, including what TOTP is and is not used for.

What users do

Open the PHANTOM VPN page, review the service, and create a personal profile with name and email.

Scan the WireGuard QR code or download the generated `.conf` file.

Open the personal profile page to check connection state, remaining trial time, and payment status.

What TOTP is for

TOTP is only for PHANTOM admin access on the admin and monitoring dashboards.

End users do not scan a TOTP code to use the VPN. They scan a WireGuard configuration QR.
